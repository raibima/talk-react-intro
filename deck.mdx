## "UI representation of data"

---

**[CodeSandbox](https://codesandbox.io/s/) is your friend**

---

## Function Component

```jsx
function Hello() {
  return <h1>Hello, Traveloka</h1>
}
```

---

## Arrow-Function Component

```jsx
const Hello = () => <h1>Hello, Traveloka</h1>
```

---

## Class Component

```jsx
class Hello extends React.Component {
  render() {
    return <h1>Hello, Traveloka</h1>
  }
}
```

---

## Component Props

```jsx
function Greeting(props) {
  return <h1>{props.text}</h1>
}

<Greeting text="Hello, Traveloka!" />
```

---

## Displaying lists

```jsx
function TodoList(props) {
  return (
    <ul>
      {props.todos.map(
        todo => <li>{todo}</li>
      )}
    </ul>
  );
}

<TodoList todos={['Buy eggs', 'Buy milk', 'Buy stuff']} />
```

---

## Forwarding Props

```jsx
function InvisibleButton(props) {
  return <button {...props} hidden />;
}

<InvisibleButton>You can't see me</InvisibleButton>
```

---

## Responding to User Interaction

```jsx
<button onClick={() => alert('Clicked!')}>Click me</button>
```

<button style={{ padding: 16, fontSize: 16, fontWeight: 'bold', borderRadius: 4, cursor: 'pointer' }} onClick={() => alert('Clicked!')}>Click me</button>
